Welcome to Cognition-Aware ABSA

This system analyzes student feedback using Aspect-Based Sentiment Analysis (ABSA) and enhances interpretation using a cognition layer based on student performance.


About the System

Traditional sentiment analysis fails to capture context. This system integrates student academic performance to better interpret feedback. High-performing students may express more critical opinions, while lower-performing students may reflect learning difficulties in their feedback.
